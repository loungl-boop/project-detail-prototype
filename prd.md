## 1. 概述

### 1.1 背景

为了满足用户对项目集管理的需求，尤迈会诊平台需要一个项目集详情页，用于展示和管理项目集的相关信息，包括基础信息、服务内容和项目清单。该页面是项目中心模块的重要组成部分，为用户提供了项目集的全面视图和管理功能。

### 1.2 产品目标

- 提供项目集基础信息的展示和编辑功能
- 展示项目集的服务内容和使用情况
- 管理项目集中的项目列表
- 提供便捷的操作入口，如服务配置、项目管理等
- 提升用户对项目集的管理效率和体验

### 1.3 产品范围

- **包含：** 项目集详情页、服务详情页、合同详情页及其相关的弹窗和抽屉组件
- **不包含：** 项目创建页、合同创建页等相关页面的具体实现

### 1.4 受益方

- **业务运营方：** 能够更有效地管理项目集，查看项目集的详细信息和服务使用情况
- **终端用户：** 能够便捷地查看和操作项目集相关信息，提高工作效率

## 3. 功能方案

### 3.1 功能架构

项目集详情页包含以下主要功能模块：

1. **项目集详情页**：
   - 基础信息模块：展示和编辑项目集的基本信息
   - 服务内容模块：展示项目集包含的服务及其使用情况
   - 项目清单模块：展示项目集中的项目列表及其状态
2. **服务详情页**：
   - 服务统计模块：展示服务的使用统计信息
   - 服务限制模块：展示和配置服务的限制策略
   - 项目分配模块：展示服务在各项目中的分配情况
3. **合同详情页**：
   - 合同信息模块：展示合同的详细信息
4. **辅助组件**：
   - 编辑项目集抽屉：用于编辑项目集的基本信息和服务配置
   - 科室限制详情弹窗：用于查看和修改科室限制
   - 数量分配弹窗：用于查看和编辑服务在各项目中的分配数量
   - 限制配置弹窗：用于查看和修改服务的限制策略
   - 修改限制配置页面：用于完整编辑服务的限制配置
   - 保存确认弹窗：用于确认保存操作
   - 退出确认弹窗：用于确认退出编辑模式操作

### 3.2 功能模块详情

#### 3.2.1 项目集详情页

[1] **功能描述：** 展示项目集的详细信息，包括基础信息、服务内容和项目清单，提供编辑、查看等操作入口。

**页面结构：**

- 左侧菜单栏：包含系统的主要导航菜单
- 顶部导航：显示当前页面路径和用户信息
- 项目标题：显示项目集名称、ID和返回按钮
- 基础信息模块：展示项目集的基本信息，支持编辑
- 服务内容模块：展示项目集包含的服务及其使用情况，支持查看服务详情
- 项目清单模块：展示项目集中的项目列表及其状态，支持查看项目详情和更多操作

**字段说明：**

| 字段名   | 类型      | 必填 | 长度限制  | 格式要求            | 说明                      |
| ----- | ------- | -- | ----- | --------------- | ----------------------- |
| 项目集名称 | 文本      | 是  | 255字符 | 字符串             | 项目集的名称                  |
| 开始时间  | 日期      | 是  | -     | YYYY-MM-DD      | 项目集的开始时间                |
| 结束时间  | 日期      | 是  | -     | YYYY-MM-DD      | 项目集的结束时间                |
| 项目负责人 | 下拉选择、多选 | 否  | 50字符  | 字符串             | 项目集的负责人                 |
| 客户名称  | 文本      | 是  | 255字符 | 字符串             | 项目集的客户名称                |
| 分配方式  | 单选      | 是  | -     | 枚举值：关联合同、手动分配   | 项目集的分配方式                |
| 合同    | 文本/链接   | 否  | 255字符 | 字符串，可点击         | 项目集关联的合同，可点击跳转到合同详情页    |
| 备注    | 文本      | 否  | 500字符 | 字符串             | 项目集的备注信息                |
| 服务名称  | 文本      | 是  | 100字符 | 字符串             | 合同中服务的名称                |
| 服务数量  | 数字      | 是  | 10位   | 正整数             | 合同中服务的总数量               |
| 已发起数量 | 数字      | 是  | 10位   | 正整数             | 已使用服务发起的工单量（踢出已取消状态数据）  |
| 已完成数量 | 数字      | 是  | 10位   | 正整数             | 服务的已完成状态数量              |
| 完成率   | 百分比     | 是  | 5位    | 0-100%          | 服务的完成比例，计算公式：已完成数量/服务数量 |
| 专家限制  | 文本      | 是  | 50字符  | 字符串             | 专家服务数量限制                |
| 患者限制  | 文本      | 是  | 50字符  | 字符串             | 患者服务次数限制                |
| 科室限制  | 文本/链接   | 是  | 50字符  | 字符串，可点击         | 科室及数量限制，可点击查看详情         |
| 项目名称  | 文本      | 是  | 100字符 | 字符串             | 项目的名称                   |
| 项目周期  | 文本      | 是  | 50字符  | 字符串             | 项目的开始和结束时间              |
| 项目进度  | 百分比     | 是  | 5位    | 0-100%          | 项目的完成进度                 |
| 状态    | 标签      | 是  | 10字符  | 枚举值：进行中、已暂停、已归档 | 项目的状态                   |
| 项目经理  | 文本      | 是  | 50字符  | 字符串             | 项目的负责人                  |

**操作：**

- 编辑：点击编辑按钮，打开右侧抽屉进行项目集信息编辑
- 查看服务详情：点击服务内容中的查看按钮，跳转到服务详情页
- 查看项目详情：点击项目清单中的查看按钮，跳转到项目详情页
- 更多操作：点击项目清单中的更多按钮，弹出操作菜单，包括编辑、服务配置、暂停、恢复、结束
- 服务分配情况：点击服务分配情况按钮，弹出数量分配弹窗
- 新建项目：点击新建项目按钮，打开新增项目页面
- 查看合同详情：点击合同链接，跳转到合同详情页

**业务流程：**

1. 用户进入项目集详情页
2. 查看基础信息、服务内容和项目清单
3. 根据需要进行编辑、查看详情等操作
4. 系统响应操作并更新页面内容

4. 调整服务分配数量
5. 点击保存按钮保存修改，或点击返回按钮放弃修改

## 8. 风险与对策

| 风险点          | 应对策略                         |
| ------------ | ---------------------------- |
| 页面加载速度慢      | 优化页面结构，减少不必要的资源加载，实现数据分页加载   |
| 弹窗交互不流畅      | 优化弹窗逻辑，确保操作响应及时，减少不必要的DOM操作  |
| 响应式设计适配问题    | 测试不同屏幕尺寸的显示效果，确保在各种设备上都能正常显示 |
| 数据量较大时表格性能问题 | 实现表格分页或虚拟滚动，优化大数据量下的性能       |
| 服务限制配置复杂     | 提供清晰的操作指引和默认值，简化配置流程         |
| 权限管理复杂       | 建立清晰的角色权限矩阵，确保权限控制的准确性和安全性   |
