## 4. 操作区域

| 操作 | 按钮样式 | 权限 | 触发结果 |
|------|----------|------|----------|
| 编辑 | 主按钮 | 有编辑权限的角色 | 跳转编辑页 |
| 删除 | 危险按钮（红色边框） | 仅管理员 | 二次确认 → 跳转列表页 |
| 自定义操作A | 次按钮 | 指定角色 | [结果说明] |

**删除二次确认：**
```
弹窗标题：确认删除？
弹窗内容：删除后数据无法恢复，确定要删除「名称」吗？
按钮：取消 / 确认删除（danger）
```

## 6. 接口契约

### 6.1 详情查询

**接口路径：** `GET /api/{module}/:id`

**响应结构：**

```json
{
  "code": 0,
  "data": {
    "id": "string",
    "name": "string",
    "status": 1,
    "creator": "string",
    "createTime": "2026-01-01 10:00:00",
    "ext": {
      "fieldX": "value",
      "fieldY": "value"
    }
  }
}
```

### 6.2 删除

**接口路径：** `DELETE /api/{module}/:id`

## 8. 边界情况

| 情况 | 处理 |
|------|------|
| 数据不存在（ID错误） | 跳转404页面或返回错误提示 |
| 无权限查看 | 提示"无权限查看" |
| 关联数据加载失败 | 显示"关联数据加载失败，点击重试" |

---

## 9. 验收标准

- [ ] 详情页返回时间 < 300ms
- [ ] 所有字段正确展示，无遗漏
- [ ] 操作日志按时间倒序排列
- [ ] 无权限操作按钮不展示
- [ ] 删除成功后跳转列表页并刷新
