## 3. 指标卡

| # | 指标名称 | 字段key | 数值格式 | 变化展示 | 数据来源 |
|---|----------|---------|----------|----------|----------|
| 1 | | | | | |
| 2 | | | | | |

**变化展示规则：**
- 上涨：`↑XX%`（绿色）
- 下降：`↓XX%`（红色）
- 持平：`持平`（灰色）
- 变化基准：与[上个周期/同期]对比

## 4. 图表说明

### 4.1 趋势图

| 字段 | 内容 |
|------|------|
| 图表类型 | 折线图 / 柱状图 |
| 维度 | 按天 / 按周 / 按月 |
| 曲线/柱子 | [曲线A]、[曲线B] |
| 交互 | 悬停显示数值，点击下钻 |

### 4.2 占比图

| 字段 | 内容 |
|------|------|
| 图表类型 | 饼图 / 环形图 |
| 分组 | [分组A]、[分组B]、[分组C] |
| 交互 | 悬停显示数值和占比 |
