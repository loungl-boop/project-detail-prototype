## 3. 筛选区

| 字段 | 筛选类型 | 说明 |
|------|----------|------|
| 状态 | 下拉多选 | 选项：全部/状态A/状态B/状态C |
| 时间范围 | 日期范围 | 默认展示近7天 |
| 关键词 | 文本输入 | 支持模糊搜索 |
| 创建人 | 下拉选择 | 人员选择器 |

**交互说明：**
- 筛选条件变化 → 自动刷新列表（防抖300ms）
- 点击「重置」→ 恢复默认筛选条件

## 5. 操作列

| 操作 | 按钮样式 | 权限 | 说明 |
|------|----------|------|------|
| 查看 | 文字按钮 | 所有人 | 跳转详情页 |

| 删除 | 文字按钮（红色） | 仅管理员 | 二次确认弹窗 |

**删除二次确认：**
```

按钮：取消 / 确认删除（danger）
```

## 7. 接口契约

### 7.1 列表查询

**接口路径：** `GET /api/{module}/list`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 默认1 |
| pageSize | int | 否 | 默认20，上限100 |
| status | string | 否 | 逗号分隔，如：`1,2` |
| startDate | string | 否 | YYYY-MM-DD |
| endDate | string | 否 | YYYY-MM-DD |
| keyword | string | 否 | 搜索关键词 |

**响应结构：**

```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": "string",
        "name": "string",
        "status": 1,
        "createTime": "2026-01-01 10:00:00"
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 20
  }
}
```

### 7.2 批量删除

**接口路径：** `POST /api/{module}/batch-delete`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ids | string[] | 是 | ID列表，最多50条 |
