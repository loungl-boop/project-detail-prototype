## 3. 列表字段

## 7. 接口契约

### 7.1 列表查询

**接口路径：** `GET /api/{module}/list`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | - | 默认1 |
| pageSize | int | - | 默认20 |
| category | string | - | 分类筛选 |
| sort | string | - | 排序字段 |
| keyword | string | - | 搜索词 |

**响应结构：**

```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": "string",
        "title": "string",
        "coverImage": "string",
        "price": 100,
        "originalPrice": 200,
        "tags": ["标签1"]
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 20,
    "hasMore": true
  }
}
```

## 8. 端适配说明

### 小程序
- 下拉刷新：`enablePullDownRefresh`
- 上拉加载：`onReachBottom` / scroll-view
- 适配：`rpx`

### H5
- 滚动：原生滚动，触摸友好
- 适配：移动端 viewport

### Web
- hover：卡片悬浮阴影
- 懒加载：IntersectionObserver
- 响应式：列表列数自适应
