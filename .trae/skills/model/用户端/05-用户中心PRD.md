# 用户端 - 用户中心 PRD

> 适用端：小程序 / H5 / Web
> 
> **端差异说明：**
> - 小程序：顶部用户卡片，固定TabBar
> - H5：移动端浏览器，顶部卡片
> - Web：桌面端，横向布局，响应式

## 5. 端适配说明

### 小程序
- 用户信息：使用 button open-type="chooseAvatar"
- TabBar：使用官方 tabBar 组件

### H5
- 头像：使用 input type="file"
- 页面滚动：原生滚动

### Web
- 布局：左侧固定导航 + 右侧内容区（桌面端）
- 响应式：移动端切换为单列布局
