## 1. 搜索区域

```
┌─────────────────────────────────────────────────────────┐
│  🔍 [输入框：输入关键词搜索...]                [搜索] [重置] │
│                                                         │
│  高级筛选（可折叠）                                        │
│  [字段A ▼] [字段B ▼] [日期范围] [字段C]                   │
└─────────────────────────────────────────────────────────┘
```

### 1.1 搜索字段

| # | 字段名 | 字段key | 搜索方式 | 说明 |
|---|--------|---------|----------|------|
| 1 | 关键词 | keyword | 模糊搜索 | 搜索范围：[字段A、字段B] |
| 2 | 状态 | status | 精确筛选 | 下拉多选 |
| 3 | 类型 | type | 精确筛选 | 下拉单选 |
| 4 | 日期范围 | startDate ~ endDate | 范围筛选 | 默认近30天 |

## 2. 搜索结果

### 2.1 结果展示

```
找到约 1,234 条结果，耗时 0.23 秒
```

| 结果类型 | 字段展示 |
|----------|----------|
| 订单 | 订单号 / 商品名称 / 买家 / 金额 / 状态 |
| 用户 | 头像 / 昵称 / 手机号 / 注册时间 |

- 搜索关键词在结果中**高亮标红**
- 匹配字段标注（如：匹配字段A）

## 4. 空结果处理

| 情况 | 展示 |
|------|------|
| 搜索无结果 | 插画 + "未找到相关结果，试试其他关键词" |

## 6. 接口契约

### 6.1 搜索接口

**接口路径：** `GET /api/search`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | ✓ | 搜索关键词 |
| type | string | - | 限定类型，如：`order,user` |
| page | int | - | 默认1 |
| pageSize | int | - | 默认20，上限50 |
| filters | JSON | - | 高级筛选条件 |

**响应结构：**

```json
{
  "code": 0,
  "data": {
    "total": 1234,
    "took": 230,
    "results": [
      {
        "type": "order",
        "id": "string",
        "title": "string",
        "snippet": "...匹配关键词...",
        "matchedFields": ["name", "desc"]
      }
    ]
  }
}
```

### 6.2 搜索建议

**接口路径：** `GET /api/search/suggest`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | ✓ | 当前输入 |
| limit | int | - | 默认10 |

- [ ] 搜索结果正确高亮关键词
