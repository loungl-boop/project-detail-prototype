# 用户端 - 详情页 PRD

> 适用端：小程序 / H5 / Web
> 
> **端差异说明：**
> - 小程序：原生页面栈导航，底部固定操作栏
> - H5：浏览器返回，底部固定操作栏
> - Web：顶部导航，支持浏览器后退，底部浮动操作区

## 3. 内容区块

### 3.1 媒体展示

| # | 类型 | 说明 |
|---|------|------|
| 主图轮播 | 图片组 | 支持横滑，1-10张 |
| 视频 | video | 点击播放（可选） |

### 3.2 价格与标题

| 字段 | 说明 |
|------|------|
| 现价 | 主题色，字号大，加粗 |
| 原价 | 灰色，删除线 |
| 折扣 | 红色背景标签（如：8折） |
| 标题 | 1-2行 |
| 副标题 | 可选，灰色小字 |
| 销量 | 可选，"已售1234件" |

### 3.3 服务/保障标签

| 标签 | 说明 |
|------|------|
| 正品保障 | 图标+文字 |
| 7天退换 | 图标+文字 |
| 极速发货 | 图标+文字 |

### 3.4 规格/参数选择

| 交互 | 说明 |
|------|------|
| 点击展开 | 展开SKU选择面板 |
| 选择规格 | 组合价格/库存联动 |
| 选中态 | 边框高亮，已选文字 |

### 3.5 店铺信息

| 字段 | 说明 |
|------|------|
| 店铺头像 | |
| 店铺名称 | |
| 进入店铺 | 点击跳转店铺主页 |

### 3.6 图文详情

| 字段 | 说明 |
|------|------|
| 内容类型 | 富文本 / HTML |
| 图片 | 需要适配最大宽度 |
| 视频 | 可内嵌视频（如有） |

## 6. 接口契约

### 6.1 详情查询

**接口路径：** `GET /api/{module}/:id`

**响应结构：**

```json
{
  "code": 0,
  "data": {
    "id": "string",
    "title": "string",
    "subtitle": "string",
    "price": 128,
    "originalPrice": 256,
    "images": ["url1", "url2"],
    "video": "url",
    "tags": ["正品保障"],
    "specs": [
      {
        "name": "颜色",
        "options": ["红色", "蓝色"]
      }
    ],
    "stock": 100,
    "sales": 1234,
    "shop": {
      "id": "string",
      "name": "string"
    },
    "content": "<html content>",
    "comments": [
      {
        "user": "string",
        "content": "string",
        "createTime": "string"
      }
    ]
  }
}
```

### 6.2 收藏/取消收藏

**接口路径：** `POST /api/favorite`

```json
{
  "id": "string",
  "action": "add | remove"
}
```

## 7. 端适配说明

### 小程序
- 页面栈：最多10层
- 分享：onShareAppMessage
- 适配：iOS/Android 全面屏

### H5
- 浏览器返回：监听 beforeunload
- URL参数：携带分享来源参数

### Web
- 桌面端：鼠标hover图片放大
- SEO：og标签填充
- 快捷键：ESC返回
