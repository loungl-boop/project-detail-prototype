## 3. 表单字段

| # | 字段名 | 字段key | 类型 | 必填 | 规则/说明 |
|---|--------|---------|------|------|-----------|
| 1 | 姓名 | name | 单行文本 | ✓ | 2-20字符 |
| 2 | 手机号 | phone | 手机号 | ✓ | 11位，格式校验 |
| 3 | 验证码 | code | 文本+倒计时 | ✓ | 6位数字，60s倒计时 |
| 4 | 地区 | region | 省市区选择器 | ✓ | 三级联动 |
| 5 | 详细地址 | address | 多行文本 | ✓ | 5-100字符 |
| 6 | 设为默认 | isDefault | 开关 | - | 默认关闭 |
| 7 | 备注 | remark | 多行文本 | - | 最多200字，字数实时显示 |

---

## 4. 验证码发送

| 字段 | 说明 |
|------|------|
| 触发时机 | 点击「获取验证码」 |
| 发送间隔 | 倒计时60s后才可再次发送 |
| 有效期 | 5分钟 |
| 错误处理 | 次数超限提示"获取次数超限，请稍后再试" |

---

## 5. 校验规则

| 字段 | 校验时机 | 错误提示 |
|------|----------|----------|
| 姓名 | 失去焦点 / 提交时 | 不能为空 / 姓名格式不正确 |
| 手机号 | 失去焦点 / 提交时 | 手机号格式不正确 |
| 验证码 | 失去焦点 / 提交时 | 验证码不能为空 / 验证码不正确 |
| 地区 | 提交时 | 请选择省市区 |
| 详细地址 | 失去焦点 / 提交时 | 详细地址不能为空 |
| 备注 | 实时 | 字数超限提示 |

**校验失败态：**
- 输入框边框变红
- 下方显示红色提示文字
- 页面滚动到第一个错误处

---

## 6. 字段联动

| 触发条件 | 联动行为 |
|----------|----------|
| 开启「设为默认」 | 关闭其他默认地址 |
| 地区选择后 | 清空详细地址（重新填写） |

---

## 7. 提交后流程

| 步骤 | 说明 |
|------|------|
| 点击提交 | 校验全部字段 |
| 校验通过 | 显示loading → 调用接口 → 成功提示 → 跳转目标页 |
| 校验失败 | 滚动到错误处，高亮错误字段 |
| 提交中 | 按钮显示loading态，禁用重复点击 |

## 8. 接口契约

### 8.1 发送验证码

**接口路径：** `POST /api/sms/send`

```json
{
  "phone": "string",
  "type": "register | login | bind"
}
```

### 8.2 提交表单

**接口路径：** `POST /api/{module}/save`

**请求参数：**

```json
{
  "name": "string",
  "phone": "string",
  "code": "string",
  "region": {
    "province": "string",
    "city": "string",
    "district": "string"
  },
  "address": "string",
  "isDefault": false
}
```

## 9. 端适配说明

### 小程序
- input适配：页面往上推，避免被键盘遮挡
- 地区选择器：使用 picker 或自定义组件

### H5
- 键盘弹出：scrollIntoView 滚动到当前输入框
- 验证码倒计时：页面可见性变化时重置

### Web
- 回车键：支持回车跳到下一个输入框
- label点击：聚焦对应输入框
- 粘贴：允许粘贴手机号
