# 会诊申请 - 选择科室功能PRD

## 1. 功能概述

本功能用于在会诊申请页面中，通过弹窗选择申请科室，并根据项目限制规则显示和限制可选科室。

## 2. 功能需求

[1] ### 2.1 选择科室弹窗

**弹窗触发**：点击申请科室字段时，从底部弹出选择科室弹窗。

**弹窗布局**：

- 弹窗标题：选择科室
- 左侧：科室分类列表
- 右侧：具体科室列表
- 关闭按钮：位于弹窗右上角

[2] ### 2.2 科室显示规则

**二级科室列表**：展示全部标准科室（状态为启用），根据科室服务限制规则，做如下显示：

| 限制规则    | 显示方式                 | 交互逻辑                    |
| ------- | -------------------- | ----------------------- |
| 不限制     | 所有科室可选               | 点击科室后更新申请科室并关闭弹窗        |
| 仅允许指定科室 | 已选择科室高亮可选，未配置科室置灰不可选 | 点击未配置科室时提示"当前项目不可申请该科室" |
| 限制特定科室  | 所有科室可选               | 点击科室后更新申请科室并关闭弹窗        |

[2] ### 2.3 可选科室规则

**科室数量限制**：

- 仅允许指定科室：判断已申请科室是否等于限制数量，如果是则在科室后显示提示"（无可用数量）"，置灰不可选
- 限制特定科室：判断已申请科室是否等于限制数量，如果是则在科室后显示提示"（无可用数量）"，置灰不可选

[2] ### 2.4 已申请科室判断规则

**判断逻辑**：

1. 未分诊工单（会诊科室为空）：取拟申请会诊科室数量=当前科室的工单数量
2. 已分诊工单（会诊科室不为空）：取会诊科室=当前科室的工单数量

**已申请科室数量**：取上述两种情况之和。

<br />

[3] ## 3. 个人中心-修改申请信息时的科室修改

### 3.1 功能概述

在个人中心的会诊详情页面，通过"更多操作"入口进入"修改申请信息"页面时，用户可以修改申请会诊科室。

### 3.2 功能需求

**科室修改流程**：

1. 用户在会诊详情页面点击右上角"更多"按钮
2. 在弹出的更多操作菜单中选择"修改申请信息"
3. 进入修改申请信息页面后，点击"申请科室"字段
4. 系统弹出选择科室弹窗，显示规则与申请会诊页面一致
5. 用户选择科室后，系统更新修改申请信息页面的科室显示

**科室显示规则**：

- 与申请会诊页面的科室显示规则完全一致，包括：
  - 科室服务限制规则（不限制、仅允许指定科室、限制特定科室）
  - 科室数量限制规则
  - 已申请科室判断规则

**交互逻辑**：

- 点击科室字段时从底部弹出选择科室弹窗
- 弹窗布局和操作方式与申请会诊页面一致
- 选择科室后更新页面显示并关闭弹窗
- 点击不可选科室时显示相应提示

### 3.3 技术实现

- 复用申请会诊页面的科室选择弹窗组件
- 确保在修改申请信息页面时，科室选择逻辑与申请会诊页面保持一致
- 确保科室状态更新及时准确
